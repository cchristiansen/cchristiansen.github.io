Project Euler Python solution 2

Problem statement

Find the sum of odd Fibonacci numbers smaller than 6 million.

Solution

Straightforward way

a = 0
b = 1
sum = 0
while b < 6000000:
	x = a+b
	a = b
	b = x
	if b%2 == 1:
		sum = sum + b
print(sum)

Other less straightforward methods do not scale well. May have to revisit to see if there is a faster method.
