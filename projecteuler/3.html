Project Euler Python solution 3

Problem statement

What is the greatest prime factor of 600851475147?

Solution

x = 600851475147

def largestprimefactor(x):
    for i in range(2,int(x**0.5)+1,1):
        if x%i == 0:
            return largestprimefactor(x/i)
    return x

print(largestprimefactor(x))

Obtaining the prime factors of large numbers is a notoriously computationally expensive problem. Expensive enough that a great deal of modern cryptography is based off of it. For this relatively small number, brute force still works. Maybe checking against a dictionary would be quicker.
